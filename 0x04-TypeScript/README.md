## Running the Code

To run the TypeScript code and build the project, follow these steps:

1. Navigate to the specific task directory (e.g., `task_0`, `task_1`, etc.).

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the project:

   ```bash
   npm run build
   ```

4. Run the code:

   ```bash
   npm start
   ```
